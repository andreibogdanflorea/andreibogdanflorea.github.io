# Introduction

This project is based around the word "productivity". The Custom File Manager is
an app that allows the user to have full control over which files they want to
add in order to access them easier later.

# Creating the application

The program was developed in C#.

Key concepts used:
* Databases
* GUI - friendly user interface; modern and minimalistic look and feel
* Usability - I tried to create straightforward features
* Classes and objects
